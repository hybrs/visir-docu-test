Welcome to VISIR-2 documentation

VISIR-2 is a numerical model designed for calculating optimal ship tracks. It serves the purpose of identifying the most efficient routes in relation to environmental factors such as the sea state, the sea currents, or the winds. The optimization objective can be voyage length, duration, or overall CO2 emissions. The numerical scheme relies on a dynamic graph-search method. Ship performance modelling is provided through a Look-Up Table (LUT).

This documentation is intended to support users in the installation and first execution of VISIR-2 code. It encompasses installation instructions, a description of the suite's structure, and guidance for operating the software. It also provides some additional information for customisations and further development. For the initial setup, we recommend focusing on Usage section of this documentation.
